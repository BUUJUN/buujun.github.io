整理 - 常用的气象专业名词

Top 发表于 2022-09-26 更新于 2022-10-17 分类于 notebook

本篇博客将整理博主目前遇到的气象相关的专业名词缩写，持续更新中...

指数

  • AAO : Antarctic Oscillation
  • AMO : Atlantic Multidecadal Oscillation
  • AO : Arctic Oscillation
  • EASM : East Asian summer monsoon index
  • ENSO : El Nino-Southern Oscillation
  • IOD : Indian Ocean Dipole
  • MJO : Madden-Julian Oscillation
  • NAO : North Atlantic Oscillation
  • PDO : Pacific Decadal Oscillation
极端降水

ETCCDI (Expert Team on Climate Change Detection and Indices)

  • PRCPTOT : Annual total PRCP in wet days
  • R95P : Annual total PRCP when PRCP > 95th percentile
  • R99P : Annual total PRCP when PRCP > 99th percentile
  • RX1day : Annual maximum 1-day precipitation
  • RX5day : Annual maximum consecutive 5-day precipitation
  • SDII : Annual total precipitation divided by the number of wet days (PRCP ≥ 1.0 mm) in the year
  • R10 : Annual count of days when PRCP ≥ 10 mm
  • R20 : Annual count of days when PRCP ≥ 20 mm
  • CDD : Maximum number of consecutive dry days with PRCP < 1 mm
  • CWD : Maximum number of consecutive wet days with PRCP ≥ 1.0 mm

方法

  • PCC : Pattern Correlation Coefficient

  • RMS : Root-Mean-Square

  • IVS : Interannual Variability Skill Score

数据

模式
  • MIP : Model Intercomparison Projects
  • PMIP : Palaeoclimate Modelling Intercomparison Project
  • DCPP : Decadal Climate Prediction Project
  • GMMIP : Global Monsoons Model Intercomparison Project
  • HighResMIP : High-Resolution Model Intercomparison Project
再分析资料
  • 20CR : 20th Century Reanalysis
  • APHRODITE : Asian precipitation—highly‑resolved observational data integration towards evaluation
  • ECMWF : European Centre for Medium-Range Weather Forecasts
  • ERA5 : The Fifth Generation ECMWF Atmospheric Reanalysis
  • JRA55 : Japanese 55-year Reanalysis
  • REGEN : Rainfall estimates on a gridded network
