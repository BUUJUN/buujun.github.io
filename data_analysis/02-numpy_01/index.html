NumPy - 语法基础 Pt.1

发表于 2022-07-31 更新于 2022-09-26 分类于 python

这篇博客主要介绍 NumPy 的数据类型和函数使用，包括 NumPy 数组的创建、索引和运算，常用的一元函数和二元函数，统计计算，线性代数计算，数组比较，随机数组的生成以及 NumPy 数组的读取与写出

Why Numpy ?

基于 NumPy 比纯 Python 处理的速度更快, 占用的内存更少
