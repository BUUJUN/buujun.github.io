Ubuntu - 系统重装

发表于 2022-09-23 更新于 2023-10-18 分类于 linux

本篇随笔将介绍 Ubuntu 系统的备份与重装

准备工作

  • 一个 8G 以上的 U盘 用作系统的启动盘
  • 下载 UltralSO 软件
  • 下载 Ubuntu 系统镜像 （清华镜像源）

制作启动盘

打开 UltralSO 软件，点击 文件 > 打开 ，选择之前下载的 Ubuntu 镜像 打开

点击 启动 > 写入硬盘映像 弹出写入硬盘映像窗口

硬盘驱动器 为插入的 U盘
映像文件 为下载的 Ubuntu 镜像的路径
写入方式 为USB-HDD+

点击 格式化 > 写入

至此，启动盘制作完成

系统备份

将系统 根目录 的所有文件打包到 /media/user/DISK 中

其中，根目录的 /proc /media /mnt /sys /lost+found 不备份

重装系统

开机按 F12 进入 BOOT ，选择 U盘启动

点击 继续 > 安装 Ubuntu > 继续 > 其他选项 进入系统分区界面

系统分区
用于挂载点分区硬盘分区大小分区类型
EFI 引导/boot/efisdb1 GB主分区
交换空间（SWAP）-sdb32 GB主分区
Ext4 日志文件系统/bootsdb2 GB逻辑分区
Ext4 日志文件系统/sdb200 GB逻辑分区
Ext4 日志文件系统/usr/localsda20 GB逻辑分区
Ext4 日志文件系统/tmpsda20 GB逻辑分区
Ext4 日志文件系统/varsda40 GB逻辑分区
Ext4 日志文件系统/optsda20 GB逻辑分区
Ext4 日志文件系统/usrsda250 GB逻辑分区
Ext4 日志文件系统/homesda600 GB逻辑分区

等待安装完成后，系统提示重启电脑，此时拔出 U盘 ，按下 Enter 即可

至此，Ubuntu 系统安装完成

系统恢复

备份重要配置文件

将配置文件复制到 /media/user/DISK 中

在 / 下解压 ubuntu_backup.tar.gz

文件恢复后，替换刚刚复制的配置文件

重启 reboot ，按 F12 ，选择 U盘 启动 ，选择 Try Ubuntu 进入系统

安装 Boot Repair 修复

安装完成后，打开 Boot Repair ，点击 Recommended repair ，等待几分钟修复完成，重新启动即可

至此，Ubuntu 系统备份、重装和恢复工作完成
