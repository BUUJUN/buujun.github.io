pd.cut( x, bins, right, labels ) : 划分面元（ binning ）

x : The input array to be binned; must be 1-D
bins : 面元, 可以是确切的面元边界, 也可以是面元数量
right : True : 左开右闭 ; False : 左闭右开
labels : 设置面元的名称
